# Sudoku solver
A simple tool to solve any sudo grid in less than 1 second.

## Build
`make`

## Usage
`./solve .8.43.9.. .3....... 2...756.. ...2..53. ..4...7.. .63..4... ..618...5 .......8. ..1.59.7.` (one argument = one line of the grid, a point is for an empty square)
